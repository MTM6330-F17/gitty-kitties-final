Our Courses

Design Foundation

Course Summary

The Design Foundations course is taught by professor Kaitlyn Tremblay. Through a hands-on approach to design theory, students learn to be objective rather than subjective about good design. Students are introduced to fundamental design rules, including principles in layout typography, color theory, user interfaces and interaction design. Modern industry-standard software is used to explore different design styles, methodologies and best practices.

Course Topics

Web Page Elements· Critique for Design· Descriptive Wireframes· Design Theories· Layout & Design· Social Validation· Color for Design· Color for Interfaces· Design Principles· Information Hierarchy· Type Deconstructed· Typography Basics· Semiotics· Simplify .

Grading

Quizzes (10%)Throughout the course, students must complete 1 quiz.Assignments (20%)Students must complete three assignments by the specified deadline. Assignment must be completed individually, and must adhere to the requirements laid out for each assignment.   Graded Exercises (5%)Throughout the course, students must successfully complete five graded exercises by the specified deadline. These exercises directly relate to the content being covered and serve as both preparations and supplementation to the content being covered in class. In-Class Exercises (10%)During each class period, there will be a demonstration of how to complete a specific task or project. Students are expected to follow along and have a completed project by the end the period. The exercises will be submitted for a grade at the end of each class period.

User Experience Design

Course Summary

The User Experience Design course is taught by professor SuCheng Lee. Going beyond the basics of creating graphics and user interfaces, students focus on how interfaces need to react to user expectations. Students apply previously learned best practices for designing effective interfaces that allow improved workflow while integrating lessons learned by gathering user feedback and usability data. Practical and effective planning for user interfaces is introduced.

Course Topics

Below you will find a list of topics that will be cover throughout the course.· Introduction To UX· User Research· Personas· Scenarios / User Journey Mapping· Information Architecture > Paper Prototype · Wireframes· Visual Design / Design Guidelines· Visual Design / Interactive Prototype Tools· Interaction Design· Animation & Transition· Usability Testing· Acceptance Testing· UX Review & Career in UX / PM Tools· Final Project Presentation· Open Lab. 

Grading

Quizzes/Q&A (20%)Throughout the course, students must complete Three quizzes and a Q&A. The topic of each quiz will be provided along with the resources.Assignments (40%)Students must complete Ten assignments by the specified deadline. Assignment must be completed individually, and must adhere to the requirements laid out for each assignment. Final Project (25%)The course has a Final Projects. The Final Project will cover all material covered thus far in class, through online courses, and through assignments and exercises.  Student must complete the final projects, by the specified deadline. Final Presentation (15%)The course has a Final Presentation.   

Server-Side Web Development

Course Summary

The Server-Side Web Development course is taught by professor Michael Eisenbraun. Up to this point the student has developed static content and focused on the client side of web development. This course we will expand on that knowledge and focus on creating dynamic web sites and application with the introduction of Server-Side Web Development.   

Course Topics

Below you will find a list of topics that will be cover throughout the course.· Creating a Local Development Environment· Basic PHP Syntax· PHP Variables· PHP Arrays· PHP Includes · PHP Conditional Statements· PHP Loops· PHP Directory Functions· PHP Image Functions· PHP Super Globals variables· PHP Custom Functions· PHP Mail· PHP Coding Standards· Databases and MySQL· SQL and CRUD· PDO and Prepared Statements· SQL Functions, Joins and Subqueries. 

Grading

In-Class Exercises (10%)During each class period, there will be a demonstration of how to complete a specific task or project using PHP and/or MySQL. Students are expected to follow along and have a completed project by the end the period. These codes will be submitted for a grade at the end of each class period. Independent Study Quizzes (10%)Throughout the course, students must complete 8 independent study quizzes. The topic of each independent study will be provided along with resources to complete the quizzes.Online Courses (10%)Throughout the course, students must successfully complete four Code School courses by the specified deadline. Exams (30%)The course has two exams, a Midterm and a Final.
